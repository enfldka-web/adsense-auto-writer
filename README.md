# 📝 AdSense Auto Writer

구글 애드센스 승인에 최적화된 블로그 글 자동 생성 도구입니다.

## ✨ 주요 기능

- **제목 추천** — 주제 입력 시 SEO 최적화된 후킹 제목 7개 추천
- **제목 직접 입력** — 원하는 제목으로 바로 글 생성
- **E-E-A-T 기반 글 작성** — 경험, 전문성, 권위성, 신뢰성을 갖춘 고품질 콘텐츠
- **6가지 구조 자동 선택** — 서론형, 비교분석형, 스토리텔링형, 가이드형, 분석형, 반박형
- **구조 중복 방지** — 이전 글과 다른 구조를 자동으로 선택
- **AI 탐지 회피** — 자연스러운 사람 말투로 작성
- **AI 이미지 프롬프트** — 글에 맞는 일러스트 생성 프롬프트 자동 제공
- **대체텍스트 자동 생성** — SEO를 위한 이미지 ALT 태그

## 🚀 사용 방법

1. `index.html` 파일을 크롬 브라우저로 열기
2. Anthropic API 키 입력 ([발급](https://console.anthropic.com/settings/keys))
3. 주제 입력 또는 제목 직접 입력
4. 카테고리, 말투, 글 길이 선택
5. 글 생성 → 티스토리에 복사 붙여넣기
